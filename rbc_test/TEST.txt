Надо решить одну из трех задач на выбор. Язык разработки ? наиболее удобный для соискателя из трех (C, PHP, Perl). Кроме исходного кода нужно указывать время, потраченное на решение.

Задача 1. Написать упрощенный аналог Unix-утилиты locate.

Первый скрипт должен рекурсивно обойти директорию, указанную ему в параметре 
и все файлы, найденные внутри этой директории записать в текстовый файл. 
Проверять наличие зацикленных symlink'ов НЕ надо.
Второй скрипт должен найти в текстовом файле (созданном первым скриптом) 
все строки, включающие подстроку из параметра.

Задача 2. Написать "пинг-понг".

Скрипт должен сделать fork(), после этого процесс-отец должен писать на 
экран 1, а процесс сын 2 так, чтобы на экране было 
12121212121212121212121212121212...., процессы должны работать без искусственных задержек.

Задача 3. Написать минимальный редакторский Web-интерфейс.

Система должна управлять статьями, у статьи есть заголовок и тело. В 
редакторском интерфейсе есть две страницы:
1. страница со списком заголовков и формой добавления статьи
2. страница открытой статьи

Статьи можно хранить где угодно. Клик по заголовку открывает статью, 
заполнение формы добавления - добавляет и опять открывает список.
